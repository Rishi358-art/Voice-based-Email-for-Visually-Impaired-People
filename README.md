# Voice Email Assistant

## Description
Voice Email Assistant is an accessible email system for visually challenged users. Manage emails hands-free using Google OAuth and speech commands.

## Features
- Read, send, and delete emails using voice
- Keyboard shortcuts for accessibility
- Speech recognition with Vosk and Web Speech API
- Google OAuth authentication
- Role-based access and audit logging

## Technologies
- Node.js + Express
- JavaScript / HTML / CSS
- Vosk Speech Recognition
- Redis caching
- PostgreSQL / Supabase
