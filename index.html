Voice-Based Email for Visually Impaired People

This application allows visually impaired users to read, send, and manage emails using voice commands. The app integrates with Gmail using Google OAuth authentication and Gmail API.

Features

  • Read emails using voice output
  • Send emails using voice commands
  • Reply to emails
  • Delete emails
  • Mark emails as read or unread
